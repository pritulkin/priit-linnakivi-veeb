# Personaalse veebilehe mall

See on kerge ja modernne isikuline veebileht, mis sobib portfoolio täitmiseks ja esimesteks tasuta töödeks.

## Failid
- `index.html` — peamine veebileht
- `styles.css` — stiilid ja kujundus
- `script.js` — mobiilne navigeerimine

## Kuidas kasutada
1. Ava `index.html` brauseris.
2. Asenda `Sinu Nimi`, `sinu.email@example.com` ja `+372 5555 5555` oma andmetega.
3. Lisa portfolio näidete asemele reaalsed projektid.

## Kuidas avaldada tasuta
- Kasuta GitHub Pages: loo uus GitHubi hoidla ja laadi `index.html`, `styles.css` ja `script.js` sinna üles.
- Ava `Settings` → `Pages` ja vali haru `main` või `master`.

## Näpunäited
- Kirjelda portfoolios iga projekti eesmärki ja sinu rolli.
- Kui soovid, saan aidata lisada ka tegeliku kontaktivormi või GitHub Pages avaldamise juhise.
